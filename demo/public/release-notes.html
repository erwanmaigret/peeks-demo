Release Notes

0.0.3

  • Added support for GLTF GLB files.
  • Autoplay GLTF animations on load.

0.0.2

  • Changed video filtering to be linear to avoid pixelated effect.
  • Improved video playback performance.
  • Auto-loop background videos.
  • Allow Drag'n'Drop of any OBJ 3d model into Peeks viewers.

0.0.1

  • Added backgroundVideo field on Page Asset: var page = new PEEKS.Asset({ backgroundVideo: '/assets/video_360.mp4' });
  • Added scene customization params on start to allow specifying fov angle: PEEKS.start(window, 'widget', { fov: 60 });
  • Set default fov angle to 60 degrees.
